
Question 1:
Step 1: Creating the main project folder, calling it “project”
Command used:
•	Mkdir project = making a folder called project
•	Cd project = I am telling it to go to the folder. 
•	Pwd = to show me where I am. 
so from now on, everything must be in the “project” folder
•	Mkdir data = making a new subfolder inside the “project” main folder and we called it “data”
•	Ls = to check if I correctly made the “data” subfolder under  “project”
•	Mkdir data/raw = making a folder inside data. 
•	Ls data = to check if raw folder has been made inside data folder. If so, it will show “raw”
•	Mkdir data/processed = same thing, we made a folder under data called “processed”
•	Ls data = it now shows the two folders under data which are: raw and processed. 
•	Mkdir analysis = making a folder under project.
•	Ls = shows us the two subfolders in project which are data and analysis
•	Mkdir analysis/fastqc = making a subfolder under analysis called fastqc
•	Mkdir analysis/bamqc = making a subfolder under analysis called bamqc
•	Ls analysis = it shows us that there are two subfolders under analysis called: fastqc and bamqc.
•	Ls = shows the three subfolders under project: data, analysis, and scripts

•	Touch data/processed/.gitkeep

•	Touch analysis/bamqc/.gitkeep
